Gestionar varias versiones de JDK en Kubuntu (o cualquier distribución
basada en Ubuntu) puede hacerse de varias maneras, aquí te describo
algunas de las más comunes:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Usando update-alternatives para gestionar versiones JDK:
\end{enumerate}

\begin{itemize}
\item
  Listar versiones instaladas:

\begin{Shaded}
\begin{Highlighting}[]
\FunctionTok{sudo}\NormalTok{ update{-}alternatives }\AttributeTok{{-}{-}config}\NormalTok{ java}
\end{Highlighting}
\end{Shaded}

  Esto te mostrará las versiones de Java disponibles y te permitirá
  elegir una como predeterminada.
\item
  Instalar una nueva versión de JDK:

  \begin{itemize}
  \item
    Puedes instalar diferentes versiones de OpenJDK usando apt:

\begin{Shaded}
\begin{Highlighting}[]
\FunctionTok{sudo}\NormalTok{ apt update}
\FunctionTok{sudo}\NormalTok{ apt install openjdk{-}8{-}jdk openjdk{-}11{-}jdk openjdk{-}17{-}jdk  }\CommentTok{\# O cualquier otra versión que desees}
\end{Highlighting}
\end{Shaded}
  \item
    Para Oracle JDK, generalmente necesitas añadir un PPA o descargarlo
    manualmente debido a las licencias:

\begin{Shaded}
\begin{Highlighting}[]
\FunctionTok{sudo}\NormalTok{ add{-}apt{-}repository ppa:linuxuprising/java}
\FunctionTok{sudo}\NormalTok{ apt update}
\FunctionTok{sudo}\NormalTok{ apt install oracle{-}java11{-}installer  }\CommentTok{\# Asegúrate de reemplazar 11 con la versión que quieras}
\end{Highlighting}
\end{Shaded}
  \end{itemize}
\item
  Configurar la versión predeterminada: Después de instalar, usa
  update-alternatives para elegir la versión predeterminada:

\begin{Shaded}
\begin{Highlighting}[]
\FunctionTok{sudo}\NormalTok{ update{-}alternatives }\AttributeTok{{-}{-}config}\NormalTok{ java}
\FunctionTok{sudo}\NormalTok{ update{-}alternatives }\AttributeTok{{-}{-}config}\NormalTok{ javac}
\end{Highlighting}
\end{Shaded}
\end{itemize}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Usando SDKMAN! para gestionar múltiples JDKs:
\end{enumerate}

\begin{itemize}
\item
  Instalar SDKMAN!:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{curl} \AttributeTok{{-}s} \StringTok{"https://get.sdkman.io"} \KeywordTok{|} \FunctionTok{bash}
\BuiltInTok{source} \StringTok{"}\VariableTok{$HOME}\StringTok{/.sdkman/bin/sdkman{-}init.sh"}
\end{Highlighting}
\end{Shaded}
\item
  Listar versiones disponibles de JDK:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{sdk}\NormalTok{ list java}
\end{Highlighting}
\end{Shaded}
\item
  Instalar una versión específica:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{sdk}\NormalTok{ install java }\OperatorTok{\textless{}}\NormalTok{version{-}id}\OperatorTok{\textgreater{}}
\end{Highlighting}
\end{Shaded}

  Donde es el identificador de la versión que deseas instalar, como
  8.0.265-zulu, 17.0.1-tem, etc.
\item
  Cambiar entre versiones:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{sdk}\NormalTok{ use java }\OperatorTok{\textless{}}\NormalTok{version{-}id}\OperatorTok{\textgreater{}}
\end{Highlighting}
\end{Shaded}
\item
  Establecer una versión como predeterminada:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{sdk}\NormalTok{ default java }\OperatorTok{\textless{}}\NormalTok{version{-}id}\OperatorTok{\textgreater{}}
\end{Highlighting}
\end{Shaded}
\item
  Eliminar una versión:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{sdk}\NormalTok{ uninstall java }\OperatorTok{\textless{}}\NormalTok{version{-}id}\OperatorTok{\textgreater{}}
\end{Highlighting}
\end{Shaded}
\item
  Actualizar una versión: SDKMAN! puede manejar actualizaciones
  automáticamente, pero para actualizar manualmente:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{sdk}\NormalTok{ upgrade java}
\end{Highlighting}
\end{Shaded}
\end{itemize}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Eliminar versiones de JDK:
\end{enumerate}

\begin{itemize}
\item
  Eliminar con apt:

\begin{Shaded}
\begin{Highlighting}[]
\FunctionTok{sudo}\NormalTok{ apt remove openjdk{-}8{-}jdk  }\CommentTok{\# Reemplaza 8 con la versión que quieras eliminar}
\end{Highlighting}
\end{Shaded}
\item
  Eliminar con SDKMAN!: Usa el comando mencionado anteriormente para
  desinstalar versiones.
\end{itemize}

Consideraciones:

\begin{itemize}
\item
  JAVA\_HOME: Después de cambiar la versión de Java, asegúrate de
  actualizar la variable de entorno JAVA\_HOME o usa SDKMAN! que puede
  manejar esto automáticamente.
\item
  Compatibilidad de aplicaciones: Algunas aplicaciones pueden requerir
  una versión específica de Java. Asegúrate de probar después de cambiar
  la versión predeterminada.
\item
  Actualizaciones de seguridad: Mantén tus JDKs actualizados para
  aplicaciones críticas debido a las actualizaciones de seguridad.
\end{itemize}

Estas herramientas y comandos te permitirán gestionar tus versiones de
JDK en Kubuntu de manera eficiente, aunque para tareas más complejas o
si tienes muchas versiones, SDKMAN! puede ser tu mejor aliado.
